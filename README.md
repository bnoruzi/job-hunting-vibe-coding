# job-hunting-vibe-coding

A personal collection of Python scripts, practice problems, and notes to support coding practice, interview preparation, and job-hunting tasks.

## Overview
This repository contains tools, small projects, and study materials organized to help you learn, practice, and streamline common job-hunting activities (resume snippets, coding challenges, interview notes, and automation scripts).

## Repository structure
- scripts/ — utility scripts and small automation tools
- challenges/ — coding problems and solutions
- examples/ — sample projects and demos
- notes/ — interview notes, tips, and checklists
- requirements.txt — Python dependencies (if applicable)

Adjust the directories above to match the repository's actual layout if needed.

## Features
- Python-based utilities and example solutions
- Self-contained challenge files for learning and interview practice
- Reusable helper scripts for common job-hunting tasks

## Requirements
- Python 3.8+
- (Optional) virtualenv or venv for dependency isolation

## Installation
1. Clone the repository:
   git clone https://github.com/bnoruzi/job-hunting-vibe-coding.git
2. Create and activate a virtual environment (recommended):
   python -m venv .venv
   source .venv/bin/activate  # macOS/Linux
   .venv\Scripts\activate     # Windows
3. Install dependencies (if any):
   pip install -r requirements.txt

## Usage
- Explore folders such as scripts/, challenges/, examples/, and notes/.
- Run Python scripts directly, for example:
   python scripts/<script_name>.py
- Use challenge files as study material or to run tests for interview preparation.
